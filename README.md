## 🔍 О проекте
Инструмент LSB-стеганографии на Python. Позволяет безопасно встраивать и извлекать произвольные файлы в/из PNG-изображений.

## ⚙️ Функционал
- Встраивание данных в PNG через модификацию младших битов (LSB)
- Автоматическое извлечение с чтением заголовка размера payload
- Интерактивное консольное меню с валидацией путей и обработкой ошибок
- Проверка совместимости размера изображения и объёма данных
